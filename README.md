# 🏠 RoomieRadar – Smart Roommate Finder Portal

## 📌 Overview
RoomieRadar is a web-based application that helps users find compatible roommates based on their preferences such as lifestyle, habits, and personal choices. The system uses a rule-based scoring algorithm to match users efficiently.

---

## 🎯 Features
- User Registration & Login
- Profile Management
- Preference Setting
- Smart Roommate Matching
- Compatibility Score Calculation
- Messaging System
- Admin Module

---

## 🧠 Matching Logic
The system uses a rule-based scoring algorithm:
- Compares user preferences field-by-field
- Assigns score for each matching preference
- Calculates compatibility percentage  
  `(matched preferences / total preferences) × 100`
- Displays best matches based on highest score

## 🛠️ Technologies Used
- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite / MySQL
- **Version Control:** Git & GitHub

## 🏗️ Architecture
The project follows **MVT (Model-View-Template)** architecture:
- **Model:** Handles database (Users, Profiles, Preferences, Matches)
- **View:** Contains business logic and processing
- **Template:** Frontend UI (HTML pages)
